About This Project

Welcome to "OnlineFoodDelivery," a passion project built from the ground up by a dedicated developer to showcase the power of modern web technologies.

The Technology

This entire application is built using only HTML, CSS and JavaScript (no frameworks). Every feature you see was coded from scratch:

  • Dynamic Menus: The restaurant and menu lists are loaded from a JSON "database," making the site fast and scalable.
  • Interactive Filtering: The "Veg" and "Non-Veg" buttons instantly filter the menu lists using JavaScript.
  • Real-Time Cart: The shopping cart adds items, calculates quantities, and updates the total price all within your browser.

The Purpose

The goal of this project was to create a responsive, real-world application that demonstrates a strong understanding of core web development skills. Thank you for visiting and testing it out!
